# Price Formatter

The script formats the price in a readable form.

# CLI usage

```bash
$ python format_price.py 1002.01
1 002.01
```

# Usage on your site

```python
from format_price import format_price

price = 1002.02
readable_price = format_price(price)
```

# Running tests

```bash
$ python tests.py
```
